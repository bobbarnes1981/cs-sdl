
This plugin adds a menu item that launches MWSPlay using the current deck.

To install:

1) Open plugins/MWSPlay/magicworkstation.ini with a text editor.
2) Set "Pictures Path #1" to your cropped card image directory.
3) Double click the file: install.reg

This is for use only if you do not have MWS installed!
