Detailed Description

FreeType 2 Binding for .NET.

Binds functions and definitions in freetype6.dll (Windows) /usr/lib/libfreetype.so.6 (Linux - specifically Fedora Core freetype install location) /Library/Frameworks/Mono.framework/Libraries/libfreetype.6.dylib (MacOSX)

The FreeType library includes the base data types and function calls to FreeType 2 to allow access to TrueType and OpenType fonts across platforms.

This is not a rendering utility and will not render fonts to the screen. It is an interface to the various font formats, and can provide either outline or bitmapped versions of font glyphs.
