\subsection{Detailed Description}
Format independent color description. \hyperlink{struct_s_d_l___color}{SDL\_\-Color} describes a color in a format independent way. You can convert a \hyperlink{struct_s_d_l___color}{SDL\_\-Color} to a pixel value for a certain pixel format using \hyperlink{_sdl_8cs_a8c6c5785c9010c08364e8abcd34648a0}{SDL\_\-MapRGB}.
