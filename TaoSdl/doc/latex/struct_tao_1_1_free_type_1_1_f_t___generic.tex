Client applications often need to associate their own data to a variety of \hyperlink{namespace_tao_1_1_free_type}{FreeType} core objects. For example, a text layout API might want to associate a glyph cache to a given size object. Most \hyperlink{namespace_tao_1_1_free_type}{FreeType} object contains a ‘generic’ field, of type \hyperlink{struct_tao_1_1_free_type_1_1_f_t___generic}{FT\_\-Generic}, which usage is left to client applications and font servers. It can be used to store a pointer to client-\/specific data, as well as the address of a ‘finalizer’ function, which will be called by \hyperlink{namespace_tao_1_1_free_type}{FreeType} when the object is destroyed (for example, the previous client example would put the address of the glyph cache destructor in the ‘finalizer’ field).
